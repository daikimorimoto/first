# TravelMemo <br>
旅行計画管理アプリです。　<br>
旅行の予定や詳細をリスト形式で管理できます。　 <br>
行き先の位置情報を簡単に確認できる機能があります。 <br>
## URL <br>

## 利用方法　<br>
旅行の計画を立てる際に行き先や時間、やりたい事などを入力し管理できます。 <br>
旅行時には計画した予定を参考に行動でき、移動前にMAP機能を使って経路確認を簡単にできます。　<br>
